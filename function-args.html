Functions Multiple Arguments

Multiple Arguments

Teminology

A parameter is what is put in the brackets () of a function

The value we pass into a parameter is an Argument

Multiple Arguments

Example

function sum (x, y) {
  console.log(x + y)
}

sum(5,5) // output 10

The order of inputting arguments is important

Example using divide

function divide(a,b) {
  console.log(a / b)
}

divide(8,2) // returns 4

However

divide(2,8) // retutns 0.25

So care is needed in the order they are submitted

If an arguemnt is missing or of the wrong type an undefined error will occur

Return statement

Built in method return values when we call them

We can use returned values in other functions

Console.log prints a value but it does not return anything, you could not save a value from a console.log into another variable.

Example

function addUp(x, y) {
  return x + y
}

let total = addUp(6,6)
console.log(total)

In the above code the return value is saved into the variable total

total can be console.log to print value

Rules about return

You can only return one thing - this doesn't mean you can't have multiple return statements, but you can only return one value

A return will end function execution, if you have code after your return it will not execute

Example

function square(x) {
  return x * x;
  console.log('All done')
}

The console.log in the above code will not run, becasue it is after the return

Note in if statements there could be more than one return, however only one of them will execute so this isn't an issue

Example

function isPurple(color) {
  if (color === 'purple') {
  return true;
    } else {
  return false;
  }
}

console.log(isPurple('purple'))

As this is a boolean and will either be yes or no it can be written as below

function isRed(color) {
return color === 'red'
}

console.log(isRed('red')) // true

The above is either red or false

This only works on simple yes or no

Placement of return is important

A misplaced return in a loop can cause it to end on it's first iteration

Example

function containsYellow(arr) {   for(let color of arr){
  if(color === 'yellow' || color === 'green'){
  return true
  }
  // return false - this placed here would exit on the first loop iteration
  }
  return false
}

console.log(containsYellow(['blue', 'red', 'yellow']))

The above inputs an array as an argument - yellow is found so true is returned
