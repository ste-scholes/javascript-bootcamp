A Few Miscellaneous JS Features

Methods

We can add functions as properties on objects

We call them methods

These can be used to group together similar functions that have to do with one another - e.g. math functions

Remember we have already used built in methods such as:
'hello.toUperCase()'

Here is an example of a method

In the below const math is a variable with a number of assigned functions or methods

These functions reside within the body of the const

They have a name assigned to them in this format add: function(x,y) this is for the addition function

const math = {
add: function(x,y){
  return x + y;
  },
multiply: function(x,y){
  return x * y;
  },
subtract: function(x,y){
  return x - y;
  }
}

console.log(math.subtract(5,10))

To call the function you need to call the variable followed by the key name

So it would be math.subtract then pass the parameters - math.subtract(5,10)

Shorhand version without key: value

This version does not require us to set up key:value pairs

const auth = {
  username: 'Stephen',
login() {
  console.log('You Logged in!')
  },
logout() {
  console.log('Goodbye')
 }
}

console.log(auth.logout())// returns Goodbye!

The above console.log calls the logout method on auth
