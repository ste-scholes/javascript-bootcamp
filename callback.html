Callback Functions

A callback function is a function that is passed into another function as an argument, which is then invoked inside the outer function

Callback function sare exteremly common in JavaScript

Normally an anonymous functions is used to call them

Example

The following code sends out an alert after 5 seconds


function grumpus (name, time) {
alert('Boo!')
}

setTimeout(grumpus, 5000);

setTimeout - built in function that calls a function or evaluates an expression after a specified number of milliseconds.

alertMe function

This function waits tree seconds then send an alert to say welcome to the user

The username is inputted with the function call

function alertMe(name) {
  setTimeout(function() {
 alert('Welcome ' + name)
 },3000);
}

alertMe('Ste') // calls function alertMe with the argument ('Ste') returns Welcome Ste in alert box
