Switch Statement

A switch statement can be used to loop through conditions, similar to a for if

The syntax is easier to understand than a for loop

The break 'breaks' out of the loop when the condition is met

let day = 7;

switch(day) {
case 1:
console.log('Monday');
break;
case 2:
console.log('Tuesday')
break;
case 3:
console.log('Wednesday')
break;
case 4:
console.log('Thursday')
break;
case 5:
console.log('Friday')
break;
case 6:
console.log('Saturday')
break;
case 7:
console.log('Sunday')
break;
default:
console.log('Invalid Day')
}

If some cases have the same output value you can chain them together
