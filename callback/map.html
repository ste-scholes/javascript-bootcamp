Array Methods

Callback Methods - Map

Arrays come with many built-in methods that accept call back functions.

Map

Generates a new array and stores the result

The original values of the array remain unchanged, they are mapped over to the new array

Example

const mynum = [20,21,22,23,24,25,26,27];

const timeTwo = mynum.map(function(num){
  return num * 2; // return is required so MAP know what value to return
})

console.log(timeTwo);

map returns a new array than needs to be captured

We do that in the variable timesTwo

Map doesn't care if you don't return the value, it runs anyway and returns undefined

Map - return object

The following code returns the array as an object of even values

This code is quite complicated so please study

const even = mynum.map(function(n){
  return{
  value: n,
  isEven: n % 2 === 0
  }
})

console.log(even)

Remember objects have key value pairs and n is the callback function

value: n

isEven: n % 2 === 0

The above returns a remainder of zero after dividing by 2, if there is a remainder, the number cannot be even!

Map Abbrivate Example

The following eaxmple takes the characters from an array and adds a full stop between the characters.

Example

const mywrd = ['asap','byob','rsvp','diy'];

const abbreve = mywrd.map(function(w) {
 return w.toUpperCase().split('').join('.');
})

The above uses the following methods below

.toUpperCase sets string to all upper case

.split Split a string into an array of substrings. The split() method is used to split a string into an array of substrings, and returns the new array. Tip: If an empty string ("") is used as the separator, the string is split between each character. Note: The split() method does not change the original string.

.join The join() method returns the array as a string. The elements will be separated by a specified separator. The default separator is comma (,).
