Array Methods

Callback Methods - Every and Some

Arrays come with many built-in methods that accept call back functions.

Every

Tests whether all elements in the array pass the provided function. It returns a boolean value.

Example

const words = ['dog', 'dig', 'log', 'bag', 'bag','wag'];

To check if all the above words have 3 characters

const threeLets = words.every(word => word.length === 3)

console.log(threeLets) // Returns true. ALL words have 3 characters

To return true Every word must have 3 characters, else it would be false

Example 2

To check every word ends with the character g

const words = ['dog', 'dig', 'log', 'bag', 'bag','wag'];

const endG = words.every(word => {
 const last = word.length -1; // -1 returns last character
 return word[last] === 'g' // evaluates the character to g
})

console.log(`All end in letter g: ${endG}`)

Some

Similar to every, but returns true if ANY of the array elements pass the test function

Returns boolean - yes or no

Some words start with d

const words = ['dog', 'dig', 'log', 'bag', 'bag','wag'];

const startsD = words.some(word => word[0] === 'd')

console.log(startsD)
