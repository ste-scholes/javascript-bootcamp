Array Methods

Callback Methods - Filter

Arrays come with many built-in methods that accept call back functions.

Filter

Creates a new array with all elements that pass the test implemented by the provided function

So for an example if you had an array of numbers and wanted to filter the even numbers out then a new array with only even numbers would be returned

It does not update, alter or mutate the original array

Example

const nums = [34,35,67,54,109,102,32,9];

Filter odd numbers from nums

const odds = nums.filter(n => n % 2 === 1)

console.log('Odd Numbers: ' + odds)

Filter even numbers from nums

const even = nums.filter(n => n % 2 === 0)

console.log('Even Numbers: ' + even)

Filter numbers over 50 from nums

const bigNums = nums.filter(n => n > 50)

console.log('Numbers Over 50: ' + bigNums)
