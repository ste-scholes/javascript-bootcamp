Array Methods

Callback Methods - Find

Arrays come with many built-in methods that accept call back functions.

Find

Find is used a lot, especially with unique identifiers

Find returns value of the first element in the array that satisfies the provided testing function

Function needs to return true or false, if true, it will return the first value and that value only. It doesn't matter if there are 10 matching values, only the first will be returned.

Example

let movies = [
  'The Fantastic Mr Fox',
  'Mr. and Mrs. Smith',
  'Mrs. Doubtfire',
  'Mr. Deeds'
];

To find the word Mrs

const movie = movies.find(movie => {   return movie.includes('Mrs');
});

console.log(movie) // Returns 'Mr. and Mrs. Smith'

To find Mrs when it appears first in array

const movie2 = movies.find(m => {
 return m.indexOf('Mrs') === 1
})

console.log(movie2) // Returns 'Mrs. Doubtfire'

Example on Object

Books array containing book objects

const books =[{
  title: 'Good Omens',
  authors: ['Terry Pratchett','Neil Gaiman'],
  rating: 4.25
},
{
  title: 'Bone: The complete Edition',
  authors: ['Jeff Smith'],
  rating: 4.42,
},
{
  title: 'American Gods',
  authors: ['Neil Gaiman'],
  rating: 4.11
},
{
  title: 'A Gentleman in Moscow',
  authors: ['Amor Towles']

  rating: 4.36
}
]})

Examples

To find a book with the rating 4.3 or above

const rating = books.find(b => b.rating >= 4.3) ;

console.log(rating)

// Returns 'Bone: The complete Edition' as this is the first instance

To find a book from author Neil Gaiman

const author = books.find(b => b.authors.includes('Neil Gaiman'))

console.log(author)

// Returns Good Omens - the first instance of a book with him as the author
