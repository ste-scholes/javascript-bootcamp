Functions

Introduction

Functions are a two step process

Define > Run

Define

function funcName() {
 // do something
 }

The above is called a function declaration or function statement

Run a function

To run a function you need the function name folllowed by ()

funcName();

Without the () you are referencing the function NOT calling it

Example

function grumpus() {
  console.log('Urgh .. you again?');
  console.log('Go away');
  console.log('Leave me alone');
}

Calling the function

console.log(grumpus())

Arguments

Arguments are values that are passed into a function

We have seen this before!

No Inputs

'hello'.toUpperCase

Arguments!

'hello'.indexOf('h') // returns 0

'hello'.indexOf('o') // returns 4

Example

The below code returns the greeting Hi along with the name defined in person

function greet(person) {
  console.log(`Hi, ${person}`);
}

Argument - Parameter

To pass a paramter into the above code

greeting('Stephen')

// returns Hi!, Stephen

Dice game

Generates a six sided dice throw

function rollDie() {
  let roll = Math.floor(Math.random() * 6) + 1;
  console.log(`Rolled: ${roll}`)
}

A function that take a paramter numRoll and loops through the value of numRoll

function numDice(numRoll) {
  for (let i = 0; i < numRoll; i++) {
  rollDie();
  }
}

The numDice function is called and the parameter is passed within the ()

numDice(2)

You can use the function in a for loop

for(let i = 0; i < 5; i++) {
  grumpus()
}

The loop would run the function 5 times

Calling a Function within another function

This code replicates rolling a six sided dice throw

Example

function rollDie() {
  let roll = Math.floor(Math.random() * 6) + 1;
  console.log(`Rolled: ${roll}`)
}

What if you needed to replicate 5 more dice like a game of Yahtzee?

We can use a loop to call the first function and loop it 5 more times

function sixDice() {
  for (let i = 0; i < 5; i++) {
  rollDie()
  }
}

To call the function

sixDice()
