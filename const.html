Arrays and Variables

Const

The value of const cannot be reassigned

const city = 'London';

const city = 'Liverpool'

console.log(city); // can't reassign const

Array values can be changed as long as the refernce remains the same

Example

const myEggs = ['brown','brown']

myEggs.push('purple')

myEggs[0] = 'green'

returns // ['green','brown','purple']

However if you try to reassign

myEggs = ['blue','pink']

this will return a Uncaught TypeError: Assignment to constant variable

const are used all the time with Arrays

Nested Array

Example

const colors = [
['red','crimson'],
['blue','navy blue'],
['green','emerald'],
['purple','orchid'],
['orange','dark orange']
]

Example 2

const animalpairs = [
['doe','buck'],
['ewe','ram'],
['peahen','peacock']
];

To access items in an nested array

console.log(animalpairs[0][0]) // returns doe

console.log(animalpairs[2][1]) // returns peacock

Example 3

const animalpairs = [
['doe',['buck','stag']],
['ewe','ram'],
['peahen',['peacock','chick']]
];

To access items in an nested array - this gets tricky

console.log(animalpairs2[0][1][0]) // returns returns buck

console.log(animalpairs2[2][1][1]) // returns chick

To update a nested array

animalpairs[0][1] = 'stag' // changes buck to stag

console.log(animalpairs[0][1] = 'stag') // using console.log
