Objects - Nested Arrays

Student Object

An object for student that has an array and another object nested within

Example

const student = {
  firstname: 'David',
  surname: 'Jones',
  strengths: ['music', 'art'], // array
  exams: { // object
    midterm: 92,
    final: 88
  }
};

Access Object within object

// say you wanted to find the average between the midterm and final exam results
You would add the midterm and final together then divide by 2
Then you would ensure midterm was added to final first by putting them in brackets

const avg = (student.exams.midterm + student.exams.final) /2;
console.log(avg)

Access Array within Object

To access art from strengths Array via variable

const strength = student.strengths[1];

console.log(strength)

Access Objects within Array

Shopping cart example

const shoppingCart = [
{
  product: 'Jenga Classic',
  price: 6.88,
  quantity: 1
},
{
  product: 'Echo Dot',
  price: 29.99,
  quantity: 3
},
{
  product: 'Fire Stick',
  price: 39.99,
  quantity: 2,
}
];

Say you want to check out how much your order would cost for the 3 Echo Dots

You would need to access the array for Echo dot which is [1]

Then you would need to times the price of the item by the quantity of the item

const costItem = shoppingCart[1].price * shoppingCart[1].quantity

console.log(costItem)

Another Example

Find cost of Fire Stick order

const fireCost = shoppingCart[2].price * shoppingCart[2].quantity

console.log('The cost of your Firestick order is ' + fireCost)

Tic Tac Toe Example

How you might use objects to structure a game of Tic Tac Toe

We use objects to create the players and an array for the playing board

Example

const game = {
player1: {
  username: 'Jasper',
  playingAs: 'X'
},
player2: {
  username:'Sooty',
  playingAs: 'O'
},
board: [
  ['O', null, 'X'],
  ['X', 'O', 'X'],
  [null,'O', 'X'],
]
}

Remember Objects are reference types just like Arrays. Variables hold a reference to the data location, not the data itself
