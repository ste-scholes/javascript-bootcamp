For Loops and Arrays

For loop - Accessing Array

To loop over an array, start at 0 and continue to the last index (.length-1)

Example

const examScore = [98,77,84,91,57,66]

You access the array using [] brackets

for(i = 0; i < examScore.length; i++) {
  console.log(i, examScore[i]);
}

Accessing using variable

for(let i = 0; i < myStudents.length; i++) {
  let student = myStudents[i]

Finding Total of all grades

Note: Putting total variable within the loop would cause the total to be reset eveytime the loop ran

let total = 0;

for(let i = 0; i < myStudents.length; i++) {
  let student = myStudents[i];
  total += student.grade;
}

console.log('Total Grades: ' + total)

Finding Average of student grades

Code doesn't need to be complicated - study this example

let total = 0;

for(let i = 0; i < myStudents.length; i++) {
  let student = myStudents[i];
  total += student.grade;
}

console.log('Avg Grades: ' + total / myStudents.length)
