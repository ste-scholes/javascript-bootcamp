Higher Order Functions

Functions that operate on/with other functions.

They can :

Accept other functions as arguments

Return a function

Example

This function callThreeTimes takes an arguemts and calls it tree times

function callThreeTimes(f) {
  f();
  f();
  f();
}

This prints out a message


function cry() {
  console.log('Boo Hoo I am so sad');
}

cry(); // Returns'Boo Hoo I am so sad'

callThreeTimes(cry) // Returns
// 'Boo Hoo I am so sad'
// 'Boo Hoo I am so sad'

Function Factories

The below code is like a factory where you can pass in variable

The function has two stages:

The first accepts a number into (num)

The second part accepts a number that will be multiplied by (num) in this function is called x

function multiplyBy(num){
  return function(x) {
  return x * num;
  }
}

The below variables are used in conjunction with the function call

They pass in the values as arguments

const triple = multiplyBy(3);
const double = multiplyBy(2);
const halve = multiplyBy(0.5)

The below function call, calls variable double which doubles the value of the number inputed

console.log(double(10)) // Returns 10
