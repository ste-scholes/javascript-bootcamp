ForEach

Array Method - Accepts callback function

Passes each value from an array into the function

Example

const numbers = [20,21,22,23,24,25,26,27];

 numbers.forEach(function(num) {
  console.log(num * 2)
})

// returns all numbers from array multiplied by 2

The callback function in the above is function(num)

Example 2

The above can also be written like the following:

const numbers = [20,21,22,23,24,25,26,27];

function printTriple (n) {
  console.log(n * 3);
})
numbers.forEach(printTriple)

Using Second Parameter - index

numbers.forEach(function(num, ind){
  console.log(ind,num);

// this prints out number and it's index

Map

Generates a new array and stores the result

The original values of the array remain unchanged, they are mapped over to the new array

Example

const mynum = [20,21,22,23,24,25,26,27];

const timeTwo = mynum.map(function(num){
return num * 2; // return is required so MAP know what value to return
})

console.log(timeTwo);

map returns a new array than needs to be captured

We do that in the variable timesTwo

Map doesn't care if you don't return the value, it runs anyway and returns undefined

Map - return object

The following code returns the array as an object of even values

This code is quite complicated so please study

const even = mynum.map(function(n){
  return{
  value: n,
  isEven: n % 2 === 0
  }
})

console.log(even)

Remember objects have key value pairs and n is the callback function

value: n

isEven: n % 2 === 0

The above returns a remainder of zero after dividing by 2, if there is a remainder, the number cannot be even!
