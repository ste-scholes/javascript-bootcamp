Nested Loop

For Loops

Example

for (let i = 1; i <= 10; i++ ){
console.log('Outer Loop',i);
for (let j = 10; j >= 0; j -= 2) {
  console.log('Inner loop',j);
  }
}

The thing to remember about this is that every time the Outer loop runs once the Inner loops completes its entire run

So the Outer loop runs once, the inner loop runs 6 times, the outer loop runs again, the Inner runs another 6 times, and so on until the Outerloop completes it's run also

Nested Loops Accessing Array

const gameBoard = [
  [4,32,8, 4],
  [64,8,32,2],
  [8,32,16,4],
  [2,8,4, 2]
];

The below code has to access the array

The first for loop collects 4,32,8,4 as variable[i]

The second loop collects 4 then sets it to [j] then colletcs 32 and sets it to [j] then 8 to [j] then 4 to [j]

let totalScore =0;
for ( let i=0; i < gameBoard.length; i++) {
  let row = gameBoard[i];
  for (let j =0; j < row.length; j++) {
  totalScore += row[j]
  }
}

console.log(totalScore)

totalScore add all the numbers of the array together
