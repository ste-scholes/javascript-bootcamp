String - Methods

Methods are actions that can be performed on strings - searching within a string, replacing parts of a string, changing upper/ lower case

Examples in code

We call Methods as follows:

thing.method() // remember the brackets!

Methods can take arguments

thing.method(arg) - We pass these in the brackets

String Length .length

var txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"; var sln = txt.length; // gives 26

indexOf

let animal = 'catfish'

animal.indexOf('cat') // returns 0 because cat starts at zero index

animal.indexOf('fish') // returns 3 because fish starts at 3

It is case sensitive so C and c are different

If what you are looking for is not found it returns -1 this can be used in conditional programing

lastIndexOf()

// method returns the index of the last occurrence of a specified text in a string:

var str = "Please locate where 'locate' occurs!";
var pos = str.lastIndexOf("locate");

Slice

returns part of a string

let games = 'football'
console.log(games.slice(4)) // returns ball because ball starts at 4

We can input 2 arguments, the first where the slice starts, the second where it ends

let games = 'football'
console.log(games.slice(0, 3)) // returns foot because ball starts at 0 and ends at 4

Replace

Replaces part of a string

let favfood = 'I like fish'
console.log(favfood.replace('fish', 'chips')) // returns 'I like chips'

It only works on the first instance - you need a regular expression to replace more or all

Search()

The search() method searches a string for a specified value and returns the position of the match:

var str = "Please locate where 'locate' occurs!";
var pos = str.search("locate");

Did you notice?

The two methods, indexOf() and search(), are equal?

They accept the same arguments (parameters), and return the same value?

The two methods are NOT equal. These are the differences:

The search() method cannot take a second start position argument.

The indexOf() method cannot take powerful search values (regular expressions).

Math methods
