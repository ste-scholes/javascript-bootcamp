Logical Operators - (&&) AND

Both side must be true in order for the whole thing to be true

Examples

1 <= 4 && 'a' === 'a' // true

9 > 10 && 9 >= 9; // false

'abc'.length === 3 && 1 + 1 === 4; // false

Example

let numNew = 11

if(numNew >= 1 && numNew <= 10) {
console.log('Number between one and ten')
} else {
console.log('Please guess number between one and ten')
}

(||) OR

If either side is true then it passes as true

true || true // true

true || false // true

false || true // true

false || false // false

Code Example:

10/2 === 5 || null // true

0 || undefined // false

Example

let age = 49;

if(age < 6 || age >= 65) {
console.log('you get in for free!')
} else {
console.log('You pay full price')
}

Another Example

let color = 'purple';

if (color === 'purple' || color === 'lilac' || color === 'orchid') {
console.log(color + ' is loverly color, good choice!')
} else {
console.log('I don\'t like the color ' + color)
}

(!) NOT

!expression returns true if the expression is false

!null // true

!(0 === 0) // false

!(3 <= 4) // false

!0 // true

!'' // true

!45 // flase

If something is already false then ! returns true

Example

let flavour = 'sasparilla';

if(flavour !== 'grape' && flavour !== 'cherry') {
console.log(`We dont have ${flavour} knobhead`)
}

// another way is to negate the entire if statement

let flavour = 'sasparilla';

if (!(flavour === 'grape' || flavour === 'cherry')) {
console.log(`We dont have ${flavour} tittyboy`)
}

Operator Precedence

let x = 7;

x == 7 || x === 3 && x > 10; // true

&& takes precedence over ||

so the above example is really

let x = 7;

x == 7 || (x === 3 && x > 10); // true

--false------------true-------

as only one side of || needs to be true it is all true

If we want it to run the way it looks use brackets ()

let x = 7;

(x == 7 || x === 3) && x > 10; // false

The exact order is !(not) && (and) || (or) - this can be changed with brackets ()

Ternary Operator

Use instead of if else on simple yes / no answer

condition ? expIfTrue : expIfFalse

Example

let numSeven = 7

numSeven === 8 ? console.log('Lucky number ' + numSeven ) : console.log('Unlucky');
