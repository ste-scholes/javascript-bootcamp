Boolean - Comparison

Comparison

All these return a boolean value // true - false

> // Greater than

< // Less than

>= // Greater or equal

<= // Less than of equal

Equality

== // equality converts both values to one type then evaluates them

Examples 7 =='7' true // it converts string to number

null == underfined // this is also true

=== // strict equality - cares both about value and type 99% of the time use this

Examples 7 == '7' false // they are different types

Non-Equality

!== // strict non-equality - just like the other stick with strict equality

10 != '10' // false

10 !== 10 // true
