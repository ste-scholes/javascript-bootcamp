GRID

To set up a grid

display: grid

Set up columns

Use fr or fraction to create colums

The below code creates 3 equal columns

grid-template-columns: 1fr 1fr;

Set up Rows

grid-template-rows: 1fr 1fr;

Set up grid-template-area

box related to the div class name/ The way the below displays is divs box1 and box2 are on the top row. While box3 stretches across both spaces on the lower row.

A row is denoted by "" so "box1 box2" are on one row.

grid-template-areas: "box1 box2"
"box3 box3"

Leaving a blank space on grid-template-area

A full stop denotes a blank cell on a grid, so on the following a 4 square grid would have box1 box2 on the top Rows with box3 nd an empty cell on the bottom row.

grid-template-areas: "box1 box2"
"box3 ."

Setting up GRID on the <main> tag

This is how you would appy grid to a page

main {
  height: 100vh;
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
 grid-template-rows: 1fr 1fr 1fr;
 grid-template-areas: "box1 box2"
  "box3 box3";
  justify-items: center;
  align-items: center;
}

Responsive colums

There are two ways to create responsive columns

Auto Fill

Auto fill will generate another new empty column when the space is availabe

grid-template-columns:repeat(auto-fill, 300px);

Auto-fit

This doesn't generate extra columns, that is all it does. We will use this more often

grid-template-columns:repeat(auto-fit, 300px);

MinMax()

minmax allows you to set a minium amount of space needed for an object, with the caviat that it can grow when extra space becomes available

In the code below the minimum value is 300px and the maximum is 1fr

grid-template-columns:repeat(auto-fit, minmax(300px, 1fr));

Image Gallery CSS Code

This is the final code to creat a responsive photo gallery in Grid

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

.gallery {
  display: grid;
  grid-template-columns:repeat(auto-fit, minmax(300px, 1fr));
  grid-column-gap: 1rem;
  grid-row-gap: 1rem;
}
.gallery img {
  width: 100%;
}
