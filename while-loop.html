While Loop - Guessing Game example

The below code generates two random numbers an evaluates them, if the match the game ends and you win!

The below variables generate a random number between 1 and 10

const target = Math.floor(Math.random() * 10)
let guess = Math.floor(Math.random() * 10)

While guess is not equal to target run the following code until it is equal

while(guess !== target) {
  console.log(`Target: ${target} Guess: ${guess}`)
  guess = Math.floor(Math.random() * 10)
}
console.log(`Snap you have a matching number! ${target} and ${guess} `)

When the code is equal it will tell you the matching numbers

console.log(target)
console.log(guess)

const targ = Math.floor(Math.random() * 10);
let guess2 = Math.floor(Math.random() * 10);

while (true) {
 if (targ === guess2) break;
  console.log(`new target: + ${targ} Guess: ${guess2}`);
  guess2 = Math.floor(Math.random() * 10);
}

console.log(`Target: ${targ} Guess: ${guess2}`);
