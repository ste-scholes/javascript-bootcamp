AJAX and AJAJ

AJAX - Asynchronous, Javascript and XML

AJAJ - Asynchronous, Javascript and JSON

AJAX - The thinking behind this is that we are communicating with a server between page loads, we can send data to or retrieve data from a server whilst the page stays the same

The communication is happening while we are on the page in the background, the information is brough into the current page

In the past the responce was in XML

Now it is usually JSON
