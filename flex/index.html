Grow

display: flex is added to the container

Add flex-grow and it's value on the inner div items

In bootstrap, the 12 column layout can be replicated by assigning flex grow to boxes as in the below

a flex-grow: 2
b flex-grow: 3
c flex-grow: 1

Shrink

display: flex is added to the container

Once again flex-shrink is added to the actual elemtents

Note: the bigger the number the more it shrinks when resized

Resize browser to see

a flex-shrink: 1
b flex-shrink: 2
c flex-shrink: 3

higher number-shrinks most

Flex wrap

Outer container add: display: flex; and flex-wrap: wrap

The inner items will wrap onto the next line when the min-width is reached

You can use wrap-reverse to revers the order in which the items wrap

Below uses wrap-reverse so the box c end up on top

box c also fills all the space because it's parent container has flex-grow: 1 applied

a flex-wrap
b flex-wrap
c flex-wrap

Flex Basis

Flex basis is similar to minumum width

You can apply flex growth to the parent div to fill available space.

The differece between min-width and flex-basis is when you get too small min-width will add scroll bar

On the other hand flex-basis will shrink the content to fit as best as possible

It is better to use flex-basis

a flex-basis: 100px
b flex-basis: 200px
c flex-basis: 300px

Flex - combining all properties

flex: 1 0 200px

This is set up on the class for the box

it interprits to grow: 1 shrink: 1 minimum width: 200px
