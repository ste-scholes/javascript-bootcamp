Flex flow

Justify content is applied to the MAIN AXIS

On the outer container flex-flow: column or row;

Changing the value or flex flow, changes the value of the axis

Flex-flow: row - main axis = row & cross axis goes down

Flex-flow: column - main axis goes down in column & cross axis = row direction
