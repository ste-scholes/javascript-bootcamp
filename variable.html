Let

let age = 49

age + 1 // 50

This does NOT change the value of age - It is still 49!

To change the value of age

age = age + 1 // value changes in variable to 50

another way to update an age by 1 year

age += 1 // 50

to remove a year

age -= 1 // 50

You can chanhe the value of const once set

const age = 49

age = age + 1 // brings back an error

myText = 1234 // this wont bring back an error
