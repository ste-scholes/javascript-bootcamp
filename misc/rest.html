A Few Miscellaneous JS Features

Rest

Before rest there was the Arguments Object

Available inside every function

It's an array like object that:

Has a length property

Does not have an array method like push/pop

Contains all arguments passed into a function

Not available inside arrow functions!

Example

function sumAll() {
let total = 0;
  for(let i=0; i < arguments.length; i++)
{
  total += arguments[i];
}
  return total;
}

console.log(sumAll(5,5,5,5)) // returns 20

You could pass any number of arguments and the function would sum them all

arguments is a function that prints all arguments passed into it

Remember arguements is not an array so you can't use reduce, pop, push etc.

Arguments used with spread ...

You can use arguments and spread to put the results into an array, this will allow you to manipulate the results as you would any array

Example

function sum() {
  const argsArray = [...arguments]
  return argsArray.reduce((total, currVal) => {
  return total + currVal
 })
}

console.log(sum(2,2,2,2,2))

Rest

Rest is used to put any passed parameter passed as arguments into an array

Rest is passed in the brackets along with a variable name that is used to store the array (...nums)

Example

function sum(...nums) {
  console.log(nums)
}

console.log(sum(1,2,3,4,5)) // returns 1,2,3,4,5 in an actual array

Example

function sum(...nums) {
  return nums.reduce((total, currVal) => {
  return total + currVal
  })
}

console.log(sum(1,1,1,3,3)) // returns 9

Remember rest returns all the remainaing arguments so that can be returned

Example

function fullName(first, last, ...titles) {
console.log('first', first)
console.log('last', last)
console.log('titles', titles)
}

console.log(fullName('Tom','Jones','III','Order of the Pheonix')) // returns
first Tom
last Jones
titles 'III','Order of the Pheonix'

The other elements from the array are collected and passed in titles

Basically rest returns everything that has been saved as a paramter and also the items not saved, these can be retured if required

Order is important ...titles must come last in the list of parameters

Arrow function using rest with implicit return

const multiply = (...nums) => (
  nums.reduce((total, currVal) => total * currVal)
)

console.log(multiply(5,5)) // returns 25
