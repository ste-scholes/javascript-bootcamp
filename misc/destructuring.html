Destructuring

A short clean syntax to unpack:

Values from Arrays

Properties from Objects

Into distinct variables

In the past the only way to get items from an array would be to save them individually to a variable

In the example below, the runners are in place order of winning

Example

const raceResults = [
  'Eluid Kipchoge',
  'Feyisa Lelisa',
  'Galen Rupp',
  'Ghirmay Ghebreslassie',
  'Alphonce Simbu',
  'Jared Ward'
];

The old way!

Useing variables to assign gold,silver, bronze

const gold = raceResults[0];
const silver = raceResults[1];
const bronse = raceResults[2];

console.log(`1st ${gold} 2nd ${silver} 3rd ${bronse}`)

Destructuring Array

Assign gold, silver, bronze

const [gold, silver, bronze] = raceResults
console.log(gold, silver, bronze)

Assign first and fourth position - use empty commas

const [first, , , fourth] = raceResults
console.log(first, fourth)

We use

Winner and the rest of the runners up using ... rest operator

const [ winner, ...runnerUp] = raceResults
console.log(winner, runnerUp)

Destructuring Array is based on position of the property

Destructuring Object

Destructuring object is based off the name of the property

Objects are targeted by keyword, not index

Example Runner

const runner = {
  first: 'Eluid',
  last: 'Kipchoge',
  country: 'Kenya',
  title: 'Elder of the Order of the Golden Heart of Kenya', }

First last name and country - seperate values with comma

const {first, last, country} = runner;
console.log(first, last, country)

To change country and title keywords to another name - nation and honorific

const {country: nation, title: honorific} = runner;
console.log(nation, honorific) // returns Kenya Elder of the Order of the Golden Heart of Kenya

Return first and last name and everything else using ... rest

const {first, last, ...details} = runner
console.log(first, last, details) // returns Eluid Kipchoge {country: "Kenya", title: "Elder of the Order of the Golden Heart of Kenya"

Remember in objects the order does not matter, keywords do!

Nested Array Objects

Here is a list of winners from 2016 Olympic Marathon

They are set as an array with the runners as objects within

const results = [{
  first: 'Eluid',
  last: 'Kipchoge',
  country: 'Kenya',
},
{
  first: 'Feyisa',
  last: 'Lilesa',
  country: 'Ethiopia',
},
{
  first: 'Galen',
  last: 'Rupp',
  country: 'United States',
}
]

To extract the country from the second object we use the following code

const [,{country}] = results
console.log(country) // Ethiopia

How does this work?

After const there is an array [ then a comma , this moves it onto the next object {country} is the keyword selected close array ]

const [{first: goldMedal}, {first: silverMedal}] = results
