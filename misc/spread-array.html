Spread Array Literals

It takes a iterable and spreads it out. It copies them over from the first array to a new array

Example

Using three seperate arrays

const cephlopods = ['dumbo octopus','humboldt squid','flamboyant cuttlefish'];

const gastorpods = ['giant african snail','banana slug' ,'variable neon slug'];

const cnidaria = ['fire coral', 'moon jelly'];

We create a new array using spread to combine cephlopods and gastorpods arrays

const mollusca = [... cephlopods, ...gastorpods ];

console.log(mollusca) // Returns ["dumbo octopus", "humboldt squid", "flamboyant cuttlefish", "giant african snail", "banana slug", "variable neon slug"

This is a new array, the originals remain unaltered

The order you call the arrays using spread is the order they will appear in the new array

You can add in other elements - here we add Garden slug to gastorpods

console.log('Garden slug',...gastorpods) // Returns Garden slug giant african snail banana slug variable neon slug

Call all the arrays elements into a new array called inverts

const inverts = [...gastorpods,...cephlopods,...cnidaria]

// Returns ["giant african snail", "banana slug", "variable neon slug", "dumbo octopus", "humboldt squid", "flamboyant cuttlefish", "fire coral", "moon jelly"]
