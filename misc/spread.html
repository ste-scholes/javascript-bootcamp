A Few Miscellaneous JS Features

Spread Function Calls

Spread syntax allows an iterable such as an array to be expanded in places where zero or more arguments (for function calls) or elements (for array literanls) are expected, or an object expression to be expanded in paces where zero or more key-value pairs (for object literals) are expected.

Example

console.log(Math.max(1,2,3,4,5,6,7,99)) //Returns 99 the highest number

Math.min will not work on arrays as it expects a number, not a sequence of numbers

const num = [1,2,3,4,5,6,7,8,99];

console.log(Math.min(num)) // Retuns NaN

Spread ...

Use spread which is three dots to acees the numbers in an array - it basically splits each item of an array into it's own part

const num = [1,2,3,4,5,6,7,8,99];

console.log(Math.max(... num)) // Returns 99

console.log(...num) // Returns all numbers from array 1 2 3 4 5 6 7 8 99

Example using functions

function giveMe4(a,b,c,d) {
  console.log('a',a)
  console.log('b',b)
  console.log('c',c)
  console.log('d',d)
}

const colours = ['red','blue','green','yellow']

This code uses the function and the above array. spread ... allows the sarray to be split out and applied to the parameters within the function

giveMe4(...colours)

Returns//
a red
b blue
c green
d yellow

The array is not changed by spread
