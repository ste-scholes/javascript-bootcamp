A Few Miscellaneous JS Features

Spread Object Literals

Make copies of objects

Example

Dog and cat objects

const feline = {
 legs: 4,
  family: 'Felidae'
};

const canine = {   family: 'Canine',
  furry: true
};

Combining dog object whilst updating with new values

const dog = {
  ...canine,
  isPet: true,
  isAdorable: true,
}

console.log(dog) // Returns "Canine", furry: true, isPet: true, isAdorable: true

dog has been combined with canine

const houseCat = {
  ...feline,
  isGrumpy: true,
  isUnpredictable: true,
  isFuckingHorrible: true,
}

The order matters, if you update one of the key values later it will take on that value

If the same key value exisits in two combined objects, it will become one value, not two seperate ones

Example

const canine = {
 family: 'Canine',
 furry: true,
  legs: 4
};

Canine has 4 legs, this is updated below in tripod

const tripod = {
  ...canine,
  legs: 3,
}

console.log(tripod) // Returns "Canine", furry: true, legs: 3

The above code overwritten the original value

HOWEVER!

If the change comes before the spread it won't take effect!

const tripod = {
  legs: 3,
  ...canine,
}

console.log(tripod) // Returns "Canine", furry: true, legs: 4

In the above code, legs was set to 3 first, then overwritten by the value brought in by spread

REMEBER Order is important if you have conflicting key values

You can use spread to ake copies of objects like you can with arrays

HOWEVER just like arrays the copy is only one level deep, you cannot clone nested objects

You can only spread an Object in another Object

const doggo = [...canine] // Return Error: spread-object.js:45 Uncaught TypeError: object is not iterable (cannot read property Symbol(Symbol.iterator))

HOWEVER

const kitty = {...[feline]} // returns legs: 4, family: "Canine", furry: true

Having an object with an array inside works as expected

Nesting Objects within Arrays also works like this

const random =[...'hello', {
  ...catDog,
}];

console.log(random) // returns ["h", "e", "l", "l", "o",]{legs: 4, family: "Canine", furry: true}
