Objects

Objects - Keys, Structure, Accessing, Commas, Square Brackets

Objects are collections of properties

Properties are key value pair

Rather than accessing data using an index, we use custom keys

Objects are like Arrays - you can assign them to a const and change the value.

The const only contains the reference to the Objects

All Keys are converted into strings!

With this in mind setting a key as a number will return an error when you try to acces it

Example

fitBitData.45 // Uncaught syntax error

Accessing Objects using keys

You access an object by using it's key

fitBitData.totalMiles // 211.7

fitBitData.avgCalorieBurn // 5755

console.log(fitBitData.avgCalorieBurn)

Commas

You don't need a comma if there is only one object

More than one object needs a comma - a trailing comma is optional

Keys are converted into strings

Explanation

In the following code the keys are numbers, however they are converted into strings automatically when assigned to being keys

const numbers = {
  100: 'one hundered',
  16: 'sixteen'
}

console.log(numbers.100) // this will not work as it looks for number 100 which doesn't exist (remember numbers are converted to strings)

console.log(numbers[100]) // this looks for the string 100 and it finds it

Square brackets

square brackets are used to access soemthing that is an invalid Javascript identifier

Explanation

const my cat ='Jasper' // this is incorrect and throws an error

Javascript allows invalid identifiers within an object key, but they throw an error when you try to access them

Similar to the problem with strings above

Example

const pets = {
  9 my cat: 'Jasper'
  dog: 'Darcehole'
}

console.log(pets.9 my cat) // throws an error key starts with a number and contains spaces

console.log(pets['9 my cat']) // even if it an invalid identifier, the square brackets allow for the string to be used and found and Jasper is returned

Accessing data using Square Brackets

As above we can use square brackets to access data

We must remember to use strings when defining

Example

const pallette = {
  red: '#eb4d4b',
  yellow: '#f9ca24',
  blue: '#30336b'
}

console.log(pallette['red']) // returns #eb4d4b

Using a variable with Square Bracket notation

Say we wanted a user to select a colour from above on a web form

We could assign the colour to a variable

let mysteryColor = 'yellow'

We access the variable mysteryColor using bracket notation on pallette object

console.log(pallete[mysteryColor] // returns #f9ca24

Bracket notation: Keyname is not valid, or when using a variable or dynamic data

REMEMBER - Trying to access something that doesn't exist return undefined
