Arrow Functions

Examples of Arrow Function syntax

Example taking one parameter

const square = (x) =>{
  return x * x
};

console.log(square(2))

Example taking two parameter

const sum = (x,y) => {
  return x + y;
}

console.log(sum(5,4))

Example 3 returns true or false depending on if a number is even

const isEven = (num) =>{
return num % 2 === 0;
}

console.log(isEven(3)) // return false because 3 is not even

Example that checks array for even numbers

const numb = [1,2,3,4,5,6,7,8];

const isEven2 = numb.map(n => {
  return n % 2 === 0;
})

console.log(isEven2)

Implicit Return

Only works in certain scenarios

Adding an implicit return means you don't have to write the return keyword

If a function take one parameter then it can be written on one line, without the need of the return key or curly braces

Example

const timesMe = n => n * n console.log(timesMe(4))

Implicit Using Arrays

const numb = [1,2,3,4,5,6,7,8];

const doubles3 = numb.map(n => n * 2)

console.log(doubles3)

The above code takes the values from the array, doubles them and returns a new array

Using with Ternary

This code used as ternary operator along with the implicit return

There are a few things to note!

const isEvens = numb.map((n) => ( // notice it uses brackets, not curly braces
n % 2 === 0 ? 'even' : 'odd' // ternary operator ? returns boolean 'even' : 'odd'
));

console.log(isEvens)
