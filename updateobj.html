Objects

Objects - Adding and Updating

Create an empty object

Example

// start with empyt object - no reviews yet

const userReview = {};

Adding values to an empty object

// add first review user is key queenBee49 with a score of value 4.0

userReview['queenBee49'] = 4.0;

// uses dot notation to add user key mrSmith78 with a score of value 3.5

userReview.mrSmith78 = 3.5

// uses dot notation to add user key colt with a score of value 5

userReview.colt = 5

Updating an Object

Example

// to update we simply do the following bracket notation (branket notation is used in variable or changing value from number to string)

userReview['colt'] = 4.7

// or using dot notation

userReview.colt = 4.9

Updating Object using operators

Example

userReview.queenBee49 += 1

// This adds 1 to the total score of queenBee49 bringing her up to 5

We can also use bracket notation

userReview['mrSmith78'] += .5

This adds .5 to mrSmith78 total score bringing him up to 4

We could also us ++

userReview['mrSmith78']++

also using dot

userReview.mrSmith78 ++

remember the = is not reuired
