for of - Nested example

Example

let magicSquare = [
  [2,7,6],
  [9,5,1],
  [4,3,8]
];

for ( let row of magicSquare) {
  let sum =0;
  for(let num of row) {
  sum += num;
  }
console.log(`${row} summed to ${sum}`)
}

The for of loops through the array and assigns the magicSquare to row variable

The next for loops takes the numbers of the array row [2,7,6] and adds them together, so 2 + 7 + 6 = 15, then loop onto the next row, as so on.

When not to use for...loop

In the below example, we use a for loop to combine the values from array words1 and array words2

const words1 = ['mail','milk','bath','black'];
const words2 = ['box','shake','tub','berry'];

for (let i =0; i < words1.length; i++) {
  console.log(`${words1[i]}${words2[i]}`);
}

// you don't have to do anything with words2 array apart from call it

Using this example and trying to do it with for loop would mean you have to try to find the index of each array in words1 and then match up the values of the words2. It is overly difficult when a for loop is a much better option

For...Of Objects

Objects will not work properly with for of, as there is no index

However you can access either keys or values using the following

console.log(Object.keys(movieReview))

console.log(Object.values(movieReview))

Remember that these use Object. keyword to access

Example

const movieReview = {
  Arrival: 9.5,
  Alien: 9,
 'Kill Bill': 8.5,
 'Star Wars': 9.7,
  Vacation: 8.5,
 'Sleepy Hollow': 9.5,
}

for (let movie of Object.keys(movieReview)) {
  console.log(movie, movieReview[movie]) // access values - see note below
}

movie is a variable so we need to use movieReview[movie]

If we need to access part of an Object and the key is a variable we need square brackets
