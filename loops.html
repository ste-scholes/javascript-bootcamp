For Loops

There are 3 seperate pieces

Initial value

When to run the loop

How to change the value each time

Example

for(let i = 1; i <=10; i++)

The above code reads- let variable i equal 1; if i is 10 or less then add 1 until 10 is reached

Example that logs hello and variable i 10 times

for(let i = 1; i <=10; i++) {
console.log('hello',i)
}

Example

Perfect squares - numbers times by themselves

for(let num = 1; num <= 20; num++) {
console.log(`${num}x${num} = ${num * num}`);
}

In the above example num is used instead of i

Subtracting using for loop

Example - count from 50 to zero by multiples of 10

for( i = 50; i >= 0; i -=10) {
console.log(i) }

Example

Count down from 200 to zero in multiples of 25

for(i = 200; i >= 0; i -=25){
console.log(i)
console.log('The loop has ended')
}

Infinite Loop

Is a loop that will run forever as its condition will never be met

Example

for(let i = 20; i >= 0; i++ {
console.log(i)
}

How to avoid

Make sure your condition is appropriate for what you are trying to do

Counting backwards

for(let i =100; i >= 0; i --) {
console.log(i)
}

Avoid using equality opperators in loops - stick with greater or less than
