Manipulating the DOM

ParentElement, children, nextSibling, previousSibling

parentElement

The parentElement refers to the parent of the current element.

Example

const firstLi = document.querySelector('li');

In the page1.html example the li parent is the ul

console.log(firstLi.parentElement)

If we use it again we get the parent of the ul - which is the body

console.log(firstLi.parentElement.parentElement)

If we use it again we get the page HTML

console.log(firstLi.parentElement.parentElement.parentElement)

If we do it one more time we get null because we have reached the top of the tree

The whole sequence is upward, from the first elements parent to the uppermost element on the page

Used to influence an element when another is clicked
