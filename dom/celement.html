createElement

Follow these steps to create and input elements on a page

This creates a new HTML object

The following code creates a h2 tag and adds content to it

Creates h2 element using createElement adds to variable newh2

const newh2 = document.createElement('h2');

Add text to the new h2

newh2.innerText = 'Its ten oclock heres Johnny'

newh2.classList.add('special') // adds class special to newh2

View results

console.log(newh2)

Its ten oclock heres Johnny

How do we get it on the page?

appendChild()

We want to get the following new h2 set in variable newh2 on the page, here's how:

const newh2 = document.createElement('h2');

Step 1 - create a variable that targets where you want it to appear

const section = document.querySelector('section');

Step 2 - use appendChild to add the variable that targets the new h2

section.appendChild(newh2)

Append always put it at the end of the element

So if you add it to a list it will appear at the end

If you add it to the body it will appear under the last element on the page within the body tag

insertBefore

If you have a UL and want to add a new list item at the top you need to follwo these steps:

1 - Target the ul tag

const parentUL = document.querySelector('ul')

2 - Create the new li HTML element with createElement

const newLI = document.createElement('li')

3 - Next add text content into the li element with innerText

newLI.innerText = 'Feed the Cat'

4 - Next target the first element where you wish this new li to proceed it.

const firstLI = document.querySelector('li.todo')

5 - Next insert the new li using inserBefore

parentUL.insertBefore(newLI, firstLI)

insertBefore take two parameters - newLI, firstLI

To add to the end - follow steps 1,2,3 then use this code

parentUL.appendChild(newLI)

To add above other elements within the ul, use the index

The code below selects the li in the second position in the index [2]

const lastUL = document.querySelectorAll('li.todo')[2]

We then insert the li using insertBefore

parentUL.insertBefore(newLI, lastUL)

insertAdjacentElement

1 - create the new element and assign to variable - in this case an i [italic tag]

const i = document.createElement('i')

2 - Next create the content that is going inbetween the italic i tags

i.innerText = 'I am Italics'

This is an alternate way of performing the tasks outlined above

3 - First target where you want the new element to go

const firstP = document.querySelector('p')

4 - Next seect where you want it to go using the correct synatx first followed by the variable of what you wish to insert

There are a number of option with insertAdjacentElement

Places element just before opening tag of parent

firstP.insertAdjacentElement('beforebegin', i)

Places element after the closing tag of parent

firstP.insertAdjacentElement('afterend', i)

Places element immediately after the opening tag of parent object

firstP.insertAdjacentElement('afterbegin', i)

Place element just before the end of parent

firstP.insertAdjacentElement('beforeend', i)
