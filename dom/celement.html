Manipulating the DOM

createElement

createElement

This creates a new HTML object

The following code creates a h2 tag and adds content to it

Creates h2 element using createElement adds to variable newh2

const newh2 = document.createElement('h2');

Add text to the new h2

newh2.innerText = 'Its ten oclock heres Johnny'

Add a class to the new h2

newh2.classList.add('special') // adds class special to newh2

View results

console.log(newh2)

// returns

Its ten oclock heres Johnny

How do we get it on the page?

appendChild()

We want to get the following new h2 set in variable newh2 on the page, here's how:

const newh2 = document.createElement('h2');

Step 1 - create a variable that targets where you want it to appear

const section = document.querySelector('section');

Step 2 - use appendChild to add the variable that targets the new h2

section.appendChild(newh2)

Append always put it at the end of the element

So if you add it to a list it will appear at the end

If you add it to the body it will appear under the last element on the page within the body tag

Add image

1 - Create a variable (newImg) that creates the new image Element

const newImg = document.createElement('img');

2 - Assign an image src to the newImg variable

newImg.src= 'https://images.unsplash.com/photo-1551582045-6ec9c11d8697?ixid=MXwxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHw%3D&ixlib=rb-1.2.1&auto=format&fit=crop&w=1301&q=80';

3 - Assign a style with a width element to control how wide the image should be

newImg.style.width='400px'; // style for image size

4 - Finally add the image into the page with the appendChild

document.body.appendChild(newImg); // add to page in body

In the above the image is appended to the body of the document

Add a link to page

1 - Create a variable to create the new link

const newLink = document.createElement('a');

2 - Create the clickable text part of the link

newLink.innerText = ' JavaScript Course. Click Here!'

3 - Create the href destination of the link

newLink.href = 'https://www.youtube.com/watch?v=PeK7jqh-LOc&t=4264s';

We now decided where to put the anchor

We place it in the first p tag on the page

4 - Create the variable for the first p tag

const firstP = document.querySelector('p');

5 - We now apend the anchor link into the first p tag on the page

firstP.appendChild(newLink);

6 - Finally we can assign a CSS class to the link - .white makes the link text white

newLink.classList.add('white')
