removeChild()

To remove a child follow these steps

1 - remove child (remove peas from list)

target parent - in a section and a ul => PARENT

const ulPeas = document.querySelector('section ul')

2 - target item you want removing - note ulPeas.querySelector => CHILD

const removePeas = ulPeas.querySelector('.special')

3 - remove element

This follows the pattern -> parent -> remove child => child

const deleted = ulPeas.removeChild(removePeas)

the removed child is returnd in the variable

console.log(deleted)

// returns <li class="special">Peas</li>
