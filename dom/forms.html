Forms - Properties and Methods

Forms and properties

Here are some examples of how to access and manipulate form data

const fields = document.querySelectorAll('input')

console.log(fields[2].value)
console.log(fields)
console.log(fields[0].value)
console.log(fields[2].checked)

Changes palceholder text on password field

fields[1].placeholder = 'Pass Word'

Finds a href with class of myLink and assigns it to variable a

const a = document.querySelector('.myLink')

Changes href on page through variable a

a.href = 'http://www.google.co.uk'

Prints result to console

console.log(a.href)

Image

Access image on page

const img = document.querySelector('img');

img.src = 'https://images.unsplash.com/photo-1561948955-570b270e7c36?ixid=MXwxMjA3fDB8MHxzZWFyY2h8Mnx8Y2F0fGVufDB8fDB8&ixlib=rb-1.2.1&auto=format&fit=crop&w=700&q=60'
