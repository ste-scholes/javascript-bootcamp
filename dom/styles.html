Manipulating styles using JavaScript

Styles

You can change CSS colour, background colour, font, hide elements using Javascript

Every element you select has a style property

To access the styles of an element use the following code

const h1 = document.querySelector('h1');
console.log(h1.style)

You can only see style defined if they are inline on the page

You cannot see them if they are on a style sheet!

You can manipulate them using Javascript

h1.style.color = "green"

Sets h1 color to green

In Javascript style names are Camel Cased

So if you wanted to restyle the background color of an item you would use:

backgroundColor

NOT

background-color

Add mulitple styles at once

Add styles in JavaScript adds them inline

So they overwrite almost any styles specified before as they are more specific

In this example we select all li on page and assign e different colour from an array to each one

Select all li on page

const allLis = document.querySelectorAll('li');

Colors Array

const colors = ['red','orange','yellow','green','blue','purple']

The code to loop through each li and assign it to a colour from the array

allLis.forEach((li, i) => {
  const color = colors[i];
  li.style.color = color;
});

the forEach has two arguments li which is all the li on the page stored in variable allLis and i. const color takes the colours from the colors array and assigns them to the index. Then li.style.color assignes an li to a color from the index.

getComputedStyle

This is used to access styles that have been passed from a style sheet

It returns all possible styles for a css property and their current values (these are mostly the default browser values)
