Manipulating the DOM

Properties and Methods

innerText

Using innerText on an element returns all the tex from within it

Example

const h1 = document.querySelector('h1');

console.log(h1.innerText) // returns my Webpage

const ul = document.querySelector('ul')

console.log(ul.innerText) returns // First Second Third

Bring back all text on page

const body = document.body.innerText;

console.log(body)

To change text of an element

To change the tect within for example an h1 tag we can do the following:

h1.innerText = "Ste's Site"

REMEMBER - innerText replaces what is there

If you used ul.innerText ='Hello' all the exisiting li items within the li would be replaced with one isnatnce of 'Hello'

It can be useful for updating a heading or a caption, when we know the content being replaced is just text and not tags etc

textContent

On first glance it seems the same as innerText, we can change the text of an element in the same way as above

However when it returns text it retains the formatting of the text, as well as any embeded tags such as script tags, it will also return bold tags etc
