Iterate over a group of objects

Using iterations

You select > loop > do something

To iterate over a group of objects use a for loop

Variable allLi selects all li on page

const allLi = document.querySelectorAll('li');

for (let i = 0; i < allLi.length; i++){
console.log(allLi[i].innerText); }

The for loop, loops over all of the elements and prints them out to console

Change all the content of the li on page

for (let i = 0; i < allLi.length; i++){
console.log(allLi[i].innerText = 'Hello World'); }

The above for loop sets all the li on the page to display the text 'Hello World'

Using a for of loop

You can use a for of loop to attain the same results as above

const allLi = document.querySelectorAll('li');

for (let li of allLi){
li.innerHTML = ' Hello World!'
}

Using this along with innerHTML allows you to set all the li tags within the page to say 'Hello World' by injecting the HTML bold tag in the string
