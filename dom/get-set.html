Manipulating the DOM

get and set - Properties and Methods

getAttribute method

Allows you to get and set attributes on elements

Example

On the form range attribute

const range = document.querySelector('input[type=range]')

Get max attribute '500'

console.log(range.getAttribute('max')) // 500

Get min

console.log(range.getAttribute('min')) // 100

Gte type

console.log(range.getAttribute('type')) // range

setAttribute

This take two attributes, the first is the name of the attribute you want tp change example min

The second is the new value you want to set (min) to.

Example

console.log(range.setAttribute('min','50'))

You can also change the type of the attribute using setAttribute

This sets the range to a radio button on the form

console.log(range.setAttribute('type','radio'))
