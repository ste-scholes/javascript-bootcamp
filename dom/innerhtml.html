Manipulating the DOM

Properties and Methods

innerHTML

Returns a string containing all the elements between the HTML

So for example h1 would return My Wepage, form returns all the elements and text held within the form tags

const form = document.querySelector('form');

console.log(form.innerHTML)

innerHTML brings back everything within the tags

It is possible to add elements into a page using innerHTML, but these must be added as a string

const title = document.querySelector('h1');

title.innerHTML = 'Hello'

// updates h1 to 'Hello'

REMEMBER innerHTML overwrites anything within the tag

So it will overwrite everything within a form tag or ul tag

You can add HTML elements this way but there are better ways

If you want to add text rather than overwrite it

const title = document.querySelector('h1');

title.innerHTML = 'Hello'

This removes the original content and sets the h1 to Hello

title.innerHTML +=' Im Stephen'

This add the words 'Im Stephen' after 'Hello' it does not replace the original rather adds to it

The difference between innerText and innerHTML is innerText will print HTML tags out as strings where innerHTML will pase them as HTML
