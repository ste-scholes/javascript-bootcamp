Manipulating the DOM

Events

All event basically follow a basic pattern

1 The Thing // A button or // input or // image

2 The Event // A click // hit return // mouse over image

3 The code to run // Change color // get search results // display image caption

Add Event Listeners

Specify the event type and a callback to run

Don't use onclick - use click as a string ('click')

Example

1 - Set variable for button

const btn = document.querySelector('button')

2 - add the event listerner

btn.addEventListener('click', function() { alert('You clicked me')})
