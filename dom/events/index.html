Manipulating the DOM

Events

All event basically follow a basic pattern

1 The Thing // A button or // input or // image

2 The Event // A click // hit return // mouse over image

3 The code to run // Change color // get search results // display image caption

Add Event Listeners

Specify the event type and a callback to run

Don't use onclick - use click as a string ('click')

Example

1 - Set variable for button

const btn = document.querySelector('button')

2 - add the event listerner

btn.addEventListener('click', function() { alert('You clicked me')})

Add a click event

Add code to change a button with ID of yes

1 - Select button

const yes = document.querySelector('#yes');

2 - Add click event to set of an alert

yes.addEventListener('click', function() {
alert('You pressed me') })

mouseover and mouseout

This code chnages the text of a button when you mouse over and again when you mouse out

1 - Select button with ID of no

const no = document.querySelector('#no');

2 - Add code for mouseover

no.addEventListener('mouseover', function() {
no.innerText = 'Stop Touching!'
});

3 - Add code for mouseout

no.addEventListener('mouseout', function() {
no.innerText = 'Click Here'
})

Without the mouseout code the change resulting from the mouseover will be permanent
