To log what key is being pressed you can use the following code

console.dir(e); // the (e) contains the info regarding what key was pressed })

keydown

1 We create an input to test keydown on

const input = document.querySelector('#username');

2 We create an addEventListener to listen for the keydown event

input.addEventListener('keydown', function(e) {
console.log('keydown')
})

The (e) is an event object that holds info about what key is pressed

keydown fires no matter what key is pressed

keypress

keypress only records an event when a character or number is generated. It will not trigger on pressing arrow keys, Shift, Ctrl or Alt as these do not generate a character when pressed. Howver return does work, even though it doesn't actually generate a character

1 We create an input to test keypress on

const input = document.querySelector('#username');

2 We create an addEventListener to listen for the keypress event

input.addEventListener('keypress', function(e) {
console.log('keypress')
})

A character or space keypress triggers the event. Example: shift key will not trigger it, nor will ctrl etc

Pressing Enter will trigger the event

Shoppinng List Exercise

View Exercise

In the following code we create a simple shopping list that will add a new list item that is adde via an input field when the return key is pressed

We use keypress to trigger the even on pressing the Enter key

1 We create the variable required to target adding items into the list via a new li tag in the ul tag

const addItemInput = document.querySelector('#addItem');

const itemsUL = document.querySelector('#items');

2 Next we write the code to perform the task

addItemInput.addEventListener('keypress', function(e){
if(e.key === 'Enter'){
  if(!this.value) return; // stops an empty string
  const newItemText = this.value; // adds new item to list - by pressing 'Enter'
  const newItem = document.createElement('li'); // we create a new li to hold the new item
  newItem.innerText = newItemText; // takes value of newItemText and adds it as text in the newItem li
  itemsUL.appendChild(newItem);// finally we add the new item
  this.value = '';// this clears the item out of the box by setting it to an empty string
} })
