Try To Click - Code Explanation

The code generates a button that randomly moves when you try to click it using a mouseover effect

Step 1 - Select the button

const btn = document.querySelector('button');

2 - Create a function that uses a randomly generated number and inserts the value into the width and height on the buttos CSS - this gives it the appearance of random movement, when the mouse goes over it.

btn.addEventListener('mouseover', function() // adds mouseover and function{
  console.log('mouse over') // Tests its working
  const height = Math.floor(Math.random() * window.innerHeight); // Generates a random number
  const width = Math.floor(Math.random() * window.innerWidth);// Uses innerWidth to stay within screen boundry
  btn.style.left = `${width}px` // Adds random width to style on btn left
  btn.style.top = `${height}px` // Adds random height to style on btn left
})

3 - Add what happens if you manage to catch the button by clicking on it

It changes the buttons inner text to 'You got me!' and the body background colour to green

btn.addEventListener('click', function() {
btn.innerText = 'You got me!';
document.body.style.backgroundColor = 'green';
})

innerWidth finds the inner width of the browser window
