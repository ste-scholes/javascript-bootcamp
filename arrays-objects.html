Arrays and Objects

Arrays

Ordered collection of values starts at index 0

List of comment

Collection of levels in a game

Song in a playlist

Example

let students = []; // empty array

Examples

let shopping = ['cheese', 'cereal', 'milk', 'ice']

console.log(shopping) // call an array

let bingo = [12,10,8,7,22]

console.log(bingo[2]) //call array on index 2 gives 8

let collection = [12, 'dog', true, null, NaN] // javascript arrays can hold different data types, other languages won't allow this

Arrays are indexed starting at zero

Acessing arrays

let colours = ['red','yellow','green', 'blue','purple']

console.log(colours[3]) // returns blue index of 3

To access the last item in an array

let colours = ['red','yellow','green', 'blue','purple']

console.log(colours[colours.length-1]) // returns purple

Modifying Arrays

Changing value in array will change original value held at that index

let shoppingList = ['Cheese', 'Milk']

shoppingList[1] ='Whole Milk' // update a value

// add to end of known list

shoppingList[2] = 'Ice Cream'

// add to end of list - unkown item length

shoppingList[shoppingList.length] = 'Tomatoes'

shoppingList[shoppingList.length] = 'Coffee'

console.log(shoppingList)

Array Methods

Push - add to end

Example

let topSongs = [
'First time I ever saw your face',
'God only knows',
'A day in the life',
'Life on Mars'
];

topSongs.push('No son of mine')

Pop - remove from end

topSongs.pop()

Shift - remove from start

Example

dishesToDo.shift() // removes item from array

// you don't need a value

Unshift - add to start

Example

dishesToDo.unshift('spoon')

// note you can chain items to add together

dishesToDo.unshift('knife','fork') // but they will retain their order as added - knife then fork

console.log(dishesToDo)

includes - look for a value NOT IE

Example

Looks for a value in an array and return true if it finds it or false if not - looks for exact direct match

let ingredients = [

'onions',

'tomatoes',

'chicken',

'peppers'

];

console.log(ingredients.includes('onions'));

//finds onion returns true

// also

console.log(ingredients.includes('onions', 2));

this looks for onions after the index starting at 2 do it returns false

Example using if

if (ingredients.includes('fish')) {

  console.log('I dont like onions')

  } else {

  console.log('im good to go')

  }

The above code can be used to easily check if something is in an array

indexOf

indexOf - just like str.indexOf

To check for blank space in string

variableName.indexOf[' '] !== -1

To check if string is present

variableName.indexOf[variable] -1

Returns the index of what your are searching for or -1 if it isn't there

Example

console.log(ingredients.indexOf('chicken')) // returns 2

console.log(ingredients.indexOf('chicken', 4)) // returns -1 (can't find it)

slice - copy portion of array

Makes a copy of theoriginal array, the original remains unchanged

Example

let animals = ['shark', 'salmon','whale,','bear','lizard','tortoise']

let swimmers = animals.slice(0,3)
 console.log(swimmers)// returns shark, salmon, whale

let mammals = animals.slice(2,4)   console.log(mammals) // returns whale, bear

let reptiles = animals.slice(4,6)
  console.log(reptiles) // returns lizard, tortoise

You can also just use the following:

let reptiles = animals.slice(4)
  console.log(reptiles)// returns lizard, tortoise - everything after index 4

let quadruped = animals.slice(-3)
  console.log(quadruped)

The above counts back three elements to give // bear, lizard, tortoise

Note

animals.slice() // makes a copy of the entire array

splice - remove/replace elements

animals.splice(1,0,'Octopus')

console.log(animals)

The above works like this (index, to delete, to add)

So it is start at index 1, delete 0, add Octopus

Example 2

animals.splice(3,2)

console.log(animals)

// the above reads at index 3, remove 2 items - removes whale, bear

To replace

animals.splice(0,2,'SHARK','OCTOPUS' )

 console.log(animals)

//The above deletes 2 items starting at index 0, then replaces them with 'SHARK','OCTOPUS'
